# Megio Panel

Extendable admin panel for your Megio apps.

## Installation

`yarn add megio-panel`

## Minimal setup

1. Import library.
   ```typescript
   import 'megio-panel'
   import 'megio-panel/styles'
   ```
2. Add element to your DOM.
   ```html
   <div id="megio-panel"></div>
   ```
